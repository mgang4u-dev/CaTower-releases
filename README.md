# CaTower

macOS Container Management GUI (Docker / Podman / Apple Container)

## Features

- Container management (start, stop, restart, delete, shell access)
- Image management (pull, delete, Docker Hub search, run)
- Volume management (create, delete, inspect)
- Network management (create, delete, inspect, connect/disconnect containers)
- Container logs & resource monitoring (CPU, memory, network)
- Multi-runtime switching (Docker Desktop, Colima, OrbStack, Podman, Apple Container)
- Docker Compose support

## Install

Download the latest `.dmg` from [Releases](https://github.com/mgang4u-dev/CaTop-releases/releases).

## System Requirements

- macOS 11.0+
- Apple Silicon (M1/M2/M3/M4)

## Tech Stack

- Tauri 2.0 (Rust)
- React + TypeScript
- Tailwind CSS
- bollard (Docker API)
